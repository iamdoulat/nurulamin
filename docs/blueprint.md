# **App Name**: Persona Folio

## Core Features:

- Home Page Showcase: An animated hero section with name, role, summary, CTA buttons (Download CV, Contact), profile photo/illustration, and background animation.
- About Page Timeline: A professional introduction with a timeline of experience, education section, and achievements/certifications.
- Dynamic Project Pages: Card grid of portfolio projects with thumbnails, titles, descriptions, and tech stack badges. Includes dynamic routes for individual projects with detailed write-ups and tech stack information.
- Service Highlights: Highlight 3-6 services (Web development, UI/UX design, etc.) with icons, titles, and descriptions.
- Blog Management: A feature to read markdown (.md/.mdx) files from /content/blogs and generate dynamic blog pages at /blog/[slug]. Includes a blog list with titles, dates, tags, and excerpts.
- Contact Form Submission: A functional contact form with name, email, and message fields, form validation, and submission handling.
- Skills self assessment: LLM-based self assessment tool: based on the applicant's experience it suggests areas to focus on when developing future skills.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) to convey professionalism and innovation.
- Background color: Light gray (#F9FAFB) to ensure a clean and modern look.
- Accent color: Emerald green (#10B981) for highlighting key elements and calls to action.
- Body text: 'Inter', a grotesque-style sans-serif, providing a neutral and modern feel for readability.
- Headline Font: 'Space Grotesk', a sans-serif, for a modern and technical look to titles.
- Use a set of consistent and professional icons to represent services, skills, and social media links.
- Employ subtle fade and slide animations powered by Framer Motion for smooth page transitions and element appearances.
